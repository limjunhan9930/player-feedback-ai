# Player Feedback Sentiment Analyzer (AI / NLP)

This is a Python-based Natural Language Processing (NLP) tool developed to analyze user reviews and player feedback. Utilizing the `TextBlob` library, the script reads live user input and mathematically categorizes the emotional polarity of the text as **Positive**, **Negative**, or **Neutral**.

### 🛠️ Technologies Used
* **Language:** Python 3
* **Libraries:** TextBlob (NLP)
* **Concepts:** Sentiment Analysis, Polarity Scoring, Continuous Input Loops

### 🚀 How to Run
1. Install the required library: `pip install textblob`
2. Run the script: `python sentiment_analyzer.py`
3. Enter a game or anime review into the console to see the AI's verdict!
